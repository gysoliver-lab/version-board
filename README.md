# 软件版本管理看板

## 使用说明

### 如何更新版本信息

编辑 `data.js` 文件，修改 `VERSION_DATA` 数组：

```javascript
const VERSION_DATA = [
    {
        version: "v1.2.23",
        date: "2026-07-10",
        content: "更新内容（用\\n换行）",
        isLatest: true  // 只有一个版本设为 true
    }
];
```

修改后保存，网站会自动更新。

### 功能特点

- ✅ 点击左侧版本，右侧显示详情
- ✅ 最新版本大字报展示
- ✅ 版本总数统计
- ✅ 更新趋势图
- ✅ 响应式设计（支持手机和电脑）
